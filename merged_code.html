Menu-Driven Program

This C++ program is a menu-driven application that provides three main functionalities:

  1. Disaster Impact Mapping
  2. Dijkstra's Algorithm
  3. Kruskal's Algorithm

1. Disaster Impact Mapping

The Disaster Impact Mapping feature allows users to manage disaster zones and their severity levels. The available operations are:

  • Add/Update Zone: Add or update a disaster zone with a severity level (High, Medium, Low).
  • Get Zone Severity: Retrieve the severity level of a specific zone.
  • Remove Zone: Delete a disaster zone from the mapping.
  • Display All Zones: List all zones along with their severity levels.

Sample Input:

Enter Zone ID: ZoneA
Enter Severity Level: High

Sample Output:

Zone ZoneA updated with severity level: High

2. Dijkstra's Algorithm

Dijkstra's algorithm finds the shortest paths from a source vertex to all other vertices in a weighted graph.

Inputs:

  • Number of vertices
  • Number of edges
  • Edges with weights
  • Source vertex

Sample Input:

Enter the number of vertices: 5
Enter the number of edges: 6
Enter edges (u v weight):
0 1 10
0 3 30
0 4 100
1 2 50
3 2 20
2 4 10
Enter the source vertex: 0

Sample Output:

Vertex    Distance from Source    Penultimate Vertex
0         0                       -1
1         10                      0
2         50                      3
3         30                      0
4         60                      2

3. Kruskal's Algorithm

Kruskal's algorithm finds the Minimum Spanning Tree (MST) of a graph. The MST connects all vertices with the minimum total weight without cycles.

Inputs:

  • Number of vertices
  • Number of edges
  • Edges with weights

Sample Input:

Enter the number of vertices: 4
Enter the number of edges: 5
Enter edges (u v weight):
0 1 10
0 2 6
0 3 5
1 3 15
2 3 4

Sample Output:

Edges in the Minimum Spanning Tree (MST):
2 - 3 : 4
0 - 3 : 5
0 - 1 : 10

Exit

The program can be exited at any time by selecting the Exit option from the main menu.

Sample Output:

Exiting program. Goodbye!
